<h3>Формат ввода:</h3>

Alphabet: abced

Not terminals: A & B & C & D & S

Start: S

S -> A & B

A -> B & A

S -> a

B -> c

C -> c

A -> a

S -> EPS


Где abced - буквы алфавита, A, B, C, D, S - не терминалы, S - стартовая, EPS - пустое слово. Нетерминалы могут быть не однобуквенными. В выводе разделяются &
